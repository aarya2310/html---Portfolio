# Aarya's Portfolio

A simple HTML portfolio website showcasing web development projects.

## Description

This is a personal portfolio website built with HTML, featuring links to various web development projects including a movie ranking page and a birthday invitation page.

## Features

- **Movie Ranking Project**: A page ranking favorite movies
- **Birthday Invite Project**: A birthday invitation webpage
- **About Page**: Information about the developer
- **Contact Page**: Contact information

## Project Structure

```
4.3 HTML Portfolio Project/
├── index.html              # Main portfolio page
├── assets/
│   └── images/             # Project preview images
│       ├── movie-ranking.png
│       └── birthday-invite.png
└── public/
    ├── about.html          # About page
    ├── contact.html        # Contact page
    ├── movie-ranking.html  # Movie ranking project
    └── birthday-invite.html # Birthday invite project
```

## How to View

Simply open `index.html` in any web browser to view the portfolio.

## Technologies Used

- HTML5
